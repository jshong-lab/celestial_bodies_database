# Celestial Bodies Database

This project is part of the freeCodeCamp Relational Database Certification.

## Project Description
In this project, I created a PostgreSQL database that models celestial bodies
such as galaxies, stars, planets, and moons.

The database includes multiple related tables with primary keys and foreign key relationships.

## Files Included
- universe.sql : PostgreSQL database dump containing the schema and data.
