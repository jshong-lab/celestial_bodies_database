## Celestial Bodies Database (freeCodeCamp)

This project was completed as part of the freeCodeCamp Relational Database Certification.


## Project Description
In this project, I created a PostgreSQL database that models celestial bodies
such as galaxies, stars, planets, and moons.

The database includes multiple related tables with primary keys and foreign key relationships to represent the hierarchical structure of celestial objects.

## Database Structure

The database includes the following tables:

- galaxy – Stores information about galaxies
- star – Contains data about stars and their associated galaxies
- planet – Stores planets that orbit stars
- moon – Contains moons that orbit planets
These tables are connected through relational constraints to maintain data integrity.

## Technologies Used

- PostgreSQL
- SQL
